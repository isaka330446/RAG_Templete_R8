# 社内問い合わせ対応ポリシー

## 回答の基本方針

問い合わせに回答する場合は、登録済み文書に記載された内容を根拠として示します。
根拠文書に明確な記載がない場合は、推測で断定せず、確認が必要であることを回答します。

## 根拠提示

重要な判断を含む回答では、根拠となる文書名、見出し、該当箇所を併記します。
複数の文書が関係する場合は、文書ごとに要点を分けて整理します。

## 対象外の質問

登録済み文書に含まれない外部制度、個別契約、個別判断が必要な質問については、対象外または追加確認が必要であることを明示します。
